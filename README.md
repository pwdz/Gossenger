# Gossenger
Go-Messenger [Client Server App]  
🚶‍♂️🚶‍♂️🚶‍♂️🚶‍♂️ **WIP**  


Currently Supported Commands:    
- **/username [username]**
- **/password [password]**
- **/changeusername [newUsername]**
- **/connect [UserID/GroupID]**
- **/send [message]**
- **/file [path]**
- **/creategp [gpName]**
- **/addmembers [UserID1] [UserID2] ...**
- **/removemembers [UserID1] [UserID2] ...** (Permission: Admin)
